## Visual Identity Standards

### Color Palette
- **Primary**: Black (#000000) - Background
- **Accent**: Purple (#8B5CF6) - Highlights, CTAs, branding
- **Secondary**: Gray variations - Text and subtle elements
- **White**: (#FFFFFF) - Text and contrast

- **Headlines**: Bold, large (4xl-8xl), strong hierarchy
- **Body**: Clean sans-serif, excellent readability
- **Emphasis**: Purple color for key terms and CTAs
